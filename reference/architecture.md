# 项目架构

## 项目结构

```
runStory/
├── src/                    # 源代码
├── tests/                  # 测试套件
├── scripts/                 # 实用脚本
├── docs/                   # 项目文档
└── CLAUDE.md               # 本文件
```

## 模块关系

- **入口点**: `src/main.py`
- **数据流**: 用户输入 → 服务层 → 数据访问层 → 响应
- **依赖注入**: 使用配置对象而非全局变量
