# 代码风格

## Python 规范

- **命名**: `snake_case` 用于变量和函数，`PascalCase` 用于类
- **导入**: 标准库优先，第三方库次之，本地模块最后
- **文档字符串**: 使用 Google 风格

## 代码质量清单

- [ ] 代码可读且命名良好
- [ ] 函数简短（<50 行）
- [ ] 文件聚焦（<800 行）
- [ ] 无深层嵌套（>4 层）
- [ ] 适当的错误处理
- [ ] 无 console.log 语句
- [ ] 无硬编码值
- [ ] 无变更（immutable 模式）
